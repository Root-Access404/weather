# Weather Dashboard

A simple, responsive weather dashboard built with HTML, CSS, and Vanilla JavaScript.

## Features
- Real-time weather data fetching using OpenWeatherMap API.
- Asynchronous data handling with `async/await`.
- Clean, responsive UI built with CSS Flexbox.

## How to run
1. Clone this repo.
2. Get an API key from [OpenWeatherMap](https://openweathermap.org/).
3. Add your key to `script.js` in the `apiKey` variable.
4. Open `index.html` using a Live Server.
